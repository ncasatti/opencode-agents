CLINGY.MD

Reference documentation for the Clingy command surface. Every endpoint and flag listed below is reachable from a default install. Deprecated entries are marked explicitly; treat anything else as stable contract.

API ENDPOINTS

VERSION 1.2.0
GET /v1/system/heartbeat

Returns real-time health metrics of the core cluster nodes.

Params: NODE_ID (UUID)
Response: 200 OK [JSON]
POST /v1/data/inject

Ingests raw telemetry stream into the persistence layer.

Params: PAYLOAD (BLOB)
Response: 202 ACCEPTED
DELETE /v1/cache/flush

Purges all ephemeral data buffers. Irreversible action.

Params: FORCE (BOOL)
Response: 204 NO_CONTENT

CLI COMMAND FLAGS

  • --verbose, -v
    Enables low-level kernel log output for debugging.
  • --target, -t [ADDR]
    Defines the remote socket address for the connection.
  • --bypass-sec
    WARNING: Disables standard security handshake verification.
  • --retry [INT]
    Maximum connection attempts before timing out.
security

SECURITY CONTRACT

All incoming streams must be signed with RSA-4096. Non-conforming packets are dropped silently.

speed

PERFORMANCE SLA

Latency overhead must not exceed 4ms per node hop for standard orchestration traffic.

emergency_home

FAILOVER PROTOCOL

Automated failover protocols trigger upon 15% packet loss across primary gateways.
