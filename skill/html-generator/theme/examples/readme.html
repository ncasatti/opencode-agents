REFERENCE ARCHITECTURE

CLINGY.MD

Clingy is a context-aware command-line framework for building modular interactive tools. It auto-discovers commands, integrates with fzf for menu selection, and ships with zero runtime dependencies beyond a POSIX shell.

The framework reads its surrounding directory tree on each invocation, discovers executables prefixed with the project sigil, and renders them into a navigable menu. Commands are added by dropping a file in place. No registration step, no configuration file.

PREREQUISITES

verified
  • RUNTIME BASH 5.0+ / ZSH 5.8+
  • DEPENDENCY FZF v0.40+
  • PLATFORM LINUX / MACOS
